# Hormur SEO Landing Pages

30 landing pages SEO optimisées pour [Hormur](https://hormur.com) — la plateforme d'intermédiation culturelle qui connecte artistes, hôtes et spectateurs pour des expériences artistiques intimes.

## Structure

- **30 pages HTML** ciblant 208 mots-clés stratégiques
- **shared-styles.css** — charte graphique Hormur (coral #EE6553, navy #323242)
- **strategie-mots-cles-hormur.csv** — base de données complète des 208 mots-clés
- **RAPPORT-SEO-HORMUR.md** — rapport diagnostique SEO complet
